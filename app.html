NEXUS App

An open source and modular desktop app.

About the NEXUS App

The NEXUS App started as a Minecraft launcher with the goal of unifying our own index "NEX", based on Zyndex, with CurseForge and Modrinth into a single program. We still maintain these functions with the a Minecraft module module by nerotvlive. However, the NEXUS App is now much more, as we've overhauled it from the ground up and refocused it to be a modular desktop program. This allows the app to serve as a central hub, a "Nexus", for everything, as it can be expanded in any "direction".

By "modular", we mean that we've equipped the NEXUS App with a public and documented API, which allows the development of add-on content for the program. The app itself is just a small core optimized to integrate the APIs and load the modules, but it optionally comes with what we consider to be the most recommendable modules available for the app.
In short: The app's modular structure allows it to do anything a module developer wants to implement. Be it launching games, executing commands, wrapping other programs, or even doing something remotely.

You, us - everyone. The NEXUS App is an open-source program with a public and documented API. Anyone who's interested in exploring the subject has the opportunity to develop modules for the app.

NEXUS App for Linux (.sh)

Requirements
bash, root (sudo) access and an internet connection

Installation
If you have curl installed, run the following command: 
curl -S https://b.nerotv.live/zyneon/nexus/app/linux/install.sh | bash
If not, install it, or run the script manually.

NEXUS App for Windows

Thank you for downloading the NEXUS App <3

Requirements
Windows 10/11

Warning
Unfortunately, our Windows installer (the .exe file) is not signed. This means your browser and computer might flag it as unsafe since they don't recognize the origin. This is because it's too expensive for us to have the .exe file signed. If you're not comfortable trusting the file, click 'More' and select the .jar file.
