Install Zyneon+

Requirements

  • A computer

  • An internet connection

Step 1: Downloading NEXUS App

If the NEXUS app is already installed, continue with step 3.
If you don't have the Zyneon Studios NEXUS Application installed on your device yet, download it from here.

If you are using Windows, it is possible that the download will be blocked because our .exe file is unsigned. However, it is safe and you can simply allow the download to continue. But if you don't want to use this, use the .jar file (requires Java 21).

Step 2: Installing NEXUS App

Please select the appropriate tab for your operating system or file. The .jar method should work for every Java compatible system.

Installing NEXUS App for Windows (.exe file)

  1. Open the previously downloaded .exe file. (double left-click it)

  2. If Windows SmartScreen blocks the file, you still need to allow the process to continue. If the file is not blocked, continue with Step 3.
    2.1 To do so, click on "More info".

    2.2 Now click on "Run anyway".

  3. Grant administrator permissions when the system asks you to do so.

  4. Follow the installer steps.

Congratulations, the Zyneon Studios NEXUS App should now be installed on your system!

Installing NEXUS App for Debian based Linux distributions (.deb file)

If you are using a distribution that is not based on Debian Linux, use the .jar file.

  1. Open the terminal.

  2. Run the following command:

    sudo apt install path/to/app/deb/file

  3. If not all dependencies are available for your system, the installation will fail, if this happens, use the .jar method.

Congratulations, the Zyneon Studios NEXUS App should now be installed on your system!

Installing NEXUS App for Java compatible systems (.jar file)

If you have a Windows system, you can also try using the .exe file.

If you are using a Debian-based Linux distribution, you can also try using the .deb file.

  1. Install Java 21 on your system if you don't already have it.
    If Java 21 is already installed, you can continue with step two.

    We recommend using Azul Zulu Java. But the normal OpenJDK from Oracle should also work.

    1.1 Download Azul Zulu 21 or Oracle OpenJDK 21 for your system.

    1.2 Install the Java version of your choice.

  2. Run the previously downloaded .jar file.

    2.1.1 If your system supports this, you can try double-clicking the .jar file. The Zyneon Studios NEXUS App should then open.
    2.1.2 If not, try starting the app from the command line.

    2.2.1 On Windows search for "Command Prompt" and open it. On macOS open "Terminal". For Linux just open your terminal of choice.

    2.2.2 Navigate to the folder where you have the .jar file.
    Example: cd /home/username/Downloads/

    2.2.3 Run the following command (Please note that you will probably need to correct the name of the .jar file if the version has changed or you have renamed the file.)

    java -jar ZyneonApplication-2024.8.jar

Step 3: Install Zyneon+ via NEXUS App

Download and install Zyneon+

  1. Open the Zyneon Studios NEXUS App.

  2. Open the Menu on the left.

  3. Click on "Discover".

  4. Select "Zyneon NEX" as source.

  5. Search for "Zyneon+".

  6. Select the Zyneon+ version of your choice and click on "Install".

  7. You're done, you can now navigate to your library and start your Zyneon+ version!
