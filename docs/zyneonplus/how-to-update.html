How to update

If you have an installed version of Zyneon+, you don't need to worry about how to update it.
If you use Zyneon+ Latest, you will always be automatically updated to the latest available Minecraft version, if you use a major version specific Zyneon+ version, for example 1.19, you will always be kept up to date with the latest version of Minecraft, e.g.: 1.19.4.

The Zyneon Studios NEXUS App does the hard work for you!
